# Data Dictionary

## Olist Customers Dataset
- `customer_id`: Unique identifier for each customer.
- `customer_unique_id`: Unique identifier linking multiple orders to the same customer.
- `customer_zip_code_prefix`: Zip code prefix of the customer's location.
- `customer_city`: City of the customer.
- `customer_state`: State of the customer.

## Olist Orders Dataset
- `order_id`: Unique identifier for each order.
- `customer_id`: Link to the customer dataset.
- `order_status`: Status of the order (e.g., delivered, canceled, processing).
- `order_purchase_timestamp`: Timestamp when the order was placed.
